# Discovery Prompt

You extract a structured product understanding from a client request.

## Inputs
- Client request (text)
- Optional industry hint
- Optional constraints

## Outputs (JSON only)
- goals[]
- users[{role, goals}]
- critical_flows[]
- unknowns[]
- risks[]

## Constraints
- must: derive only from the client request; never invent regulated claims
- must_not: invent modules outside evidence; skip DNA generation here
- should: flag unknowns explicitly

## Handoff
Emit `handoffs/discovery-to-requirements` payload for the next step.
