## Playbook maturity

| Level | Meaning |
|-------|---------|
| Planned | Index entry only |
| Seed | Discovery questions + modules/risks |
| Validated | Used in ≥1 OR dogfood end-to-end |
| Proven | Used in real client delivery with feedback → lessons |

Promote levels only with evidence (Constitution §9).

---

## Each playbook must include

1. Discovery question bank  
2. Typical workflows  
3. Required / recommended / advanced capabilities  
4. Compliance notes  
5. Integrations  
6. Risks  
7. Upsells  
8. Benchmark refs (patterns only)  
9. Knowledge graph links (when available)
