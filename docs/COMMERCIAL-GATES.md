# Commercial Quality Gates

**Before a proposal is finalized and Project DNA is issued for manufacturing.**

Aligned with Constitution §5–§6 and Delivery Governance.

---

## Pre-proposal / pre-DNA checklist

| # | Gate | Pass criteria |
|---|------|----------------|
| 1 | Discovery confidence | Overall ≥ threshold (default **90%**) |
| 2 | Business objectives | Defined and accepted by consulting trail |
| 3 | Scope complete | Essential capabilities listed; out-of-scope explicit |
| 4 | Risks identified | Risk register non-empty for industry-sensitive work |
| 5 | Features justified | Each recommended feature has Decision Intelligence provenance |
| 6 | Pricing reviewed | Commercial package generated; deposit terms clear |
| 7 | Legal prepared | SOW + MSA drafts present (counsel review as required) |
| 8 | Governance recorded | Approvals path documented (internal → client → deposit) |

**Only then** generate / release Project DNA for the Product Factory.

---

## Fail closed

If any gate fails:

- Do **not** call factory handoff  
- Continue discovery, analysis, or commercial revision  
- Record blocker in engagement governance history
